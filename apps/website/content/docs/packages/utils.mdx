## Overview

The `@budgetbuddyde/utils` package is a collection of utility functions and helpers used in the various services and applications. It provides a set of reusable functions that can be used across different projects to avoid code duplication and improve maintainability.

```package-install
npm i @budgetbuddyde/utils
```

## Publishing

The package is published automatically when changes are made to the `main` branch. Publishing takes place via Concourse CI within the [bb-utils](https://ci.tools.tklein.it/teams/main/pipelines/bb-utils?group=all) workflow.
