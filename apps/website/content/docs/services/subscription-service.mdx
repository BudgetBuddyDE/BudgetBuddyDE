## Overview

<Callout title="Note" type="warn">
  This “service” is no longer supported, as subscriptions (recurring payments)
  are now processed directly in the Pocketbase instance as implemented in the
  commit
  [`6dec29e779a4321be07d3d4dc1db96dccefe34bb`](https://github.com/BudgetBuddyDE/Pocketbase/commit/6dec29e779a4321be07d3d4dc1db96dccefe34bb)
</Callout>

The Subscription Service seamlessly manages recurring payments for users within the app. It automatically
executes the stored payments and generates transactions that users can view within the app.

## Installation

<Steps>
   <Step>
      ### 1. Clone the repository

      ```bash

      git clone git@github.com:BudgetBuddyDE/Subscription-Service.git
      ```

   </Step>
   
   <Step>
      ### 2. Install dependencies

      ```bash
      npm install
      ```

   </Step>

   <Step>
      ### 3. Set all required environment variables

      ```bash
      cp .env.example .env
      ```

      Adjust the values in the `.env` file according to your setup.

   </Step>

   <Step>
      ### 4. Start the application
      ```bash
      npm run start
      ```
   </Step>
</Steps>

## Docker

<Callout title="Attention">
You may need to sign into the Github Image Registry by using `echo <GH_PAT> | docker login ghcr.io -u <GH_USERNAME> --password-stdin`
</Callout>

<Steps>
   <Step>
      ### 1. Pull the image

      ```bash
      docker pull ghcr.io/budgetbuddyde/subscription-service:latest
      ```

   </Step>

   <Step>
      ### 2. Start an container
      ```bash
      docker run --env-file .env ghcr.io/budgetbuddyde/subscription-service:latest
      ```
   </Step>
</Steps>
