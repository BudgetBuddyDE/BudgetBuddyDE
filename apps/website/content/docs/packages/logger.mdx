## Overview

<Callout>
`@budgetbuddyde/logger` is based on [`@tklein1801/logger.js`](https://github.com/tklein1801/logger.js)
</Callout>

The `@budgetbuddyde/logger` package is a centralized logging utility designed to standardize logging practices across all BudgetBuddy applications and services. It provides a consistent interface for logging messages, errors, and other relevant information, making it easier to monitor and debug applications.

```package-install
npm i @budgetbuddyde/logger
```

## Publishing

The package is published automatically when changes are made to the `main` branch. Publishing takes place via Concourse CI within the [bb-logger](https://ci.tools.tklein.it/teams/main/pipelines/bb-logger?group=all) workflow.
