## Overview

<Callout title="Important Information" type="warn">
  The Stock Service is not publicly available yet. It is currently in development and will be
  released soon.
</Callout>

The Stock Service is responsible for retrieving current prices of securities and providing detailed information
about assets. Users can search for specific assets through the service and then access detailed information
about the asset. Additionally, registered users of the app can create a watchlist through the service
and manage positions of specific securities to stay informed about the current market value of their
positions. Users are also able to view future dividends of a specific asset.

### Features

<Accordions type="single" orientation="vertical">
  <Accordion title="Manage stock positions">
    Users can create a watchlist to track current price information for selected securities.
  </Accordion>
  <Accordion title="Retrieve quotes">
    Get the latest security prices for assets listed on different exchanges.
  </Accordion>
  <Accordion title="Real-time price updates">
    Stay updated with real-time price changes by subscribing to a websocket endpoint for specific
    securities. Users can subscribe and unsubscribe from live price updates and receive updates for
    the securities they have subscribed to.
  </Accordion>
  <Accordion title="Dividends">View upcoming dividend payouts for owned positions.</Accordion>
  <Accordion title="Watchlist">
    Create a personalized watchlist to easily monitor the prices of selected assets.
  </Accordion>
  <Accordion title="Metal quotes">
    Retrieve current daily prices for gold, platinum, and silver using the
    [metalpriceapi](https://metalpriceapi.com/).
  </Accordion>
  <Accordion title="Caching with Redis">
    Caching the metal prices to transmit them faster to the client and reduce the costs of using
    external APIs.
  </Accordion>
</Accordions>

## Getting started

<Steps>
   <Step>
      ### 1. Clone the repository
   
      ```bash
      git clone https://github.com/BudgetBuddyDE/BudgetBuddyDE.git
      cd services/stock-service
      ```
   </Step>

   <Step>
      ### 2. Install the dependencies

      ```bash
      npm install
      ```

   </Step>

   <Step>
      ### 3. Set all required environment variables

      ```bash
      cp .env.example .env
      ```

      Adjust the values in the `.env` file according to your setup.
    </Step>

   <Step>
      ### 4. Start the application
   
      ```bash
      npm run start
      # or in developer mode
      npm run dev
      ```
   </Step>
</Steps>
