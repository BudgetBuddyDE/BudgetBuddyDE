The `@budgetbuddyde/types` package is a collection of types and schemas used in the various services and applications.

```package-install
npm i @budgetbuddyde/types
```

## Publishing

The package is published automatically when changes are made to the `main` branch. Publishing takes place via Concourse CI within the [bb-types](https://ci.tklein.it/teams/budgetbuddyde/pipelines/types) workflow.
