# Getting started

BudgetBuddy helps you track income and expenses, manage recurring payments, set budgets, and understand your financial activity.

## Recommended path

1. Open a BudgetBuddy instance or complete the [self-hosting guide](self-hosting.md).
2. Create an account and sign in.
3. Configure categories and payment methods.
4. Add your first income or expense.
5. Create a budget and review the dashboard.

The detailed feature guides will be added here as each workflow is documented.

## Current capabilities

- Dashboard with financial statistics and upcoming recurring payments
- Transaction management for income and expenses
- Recurring payments
- Category budgets and progress indicators
- Custom categories and payment methods
- Table filtering with shareable URL state

Planned features are tracked separately and are not presented as available functionality.
